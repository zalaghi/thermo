Security Notes

Keep agents private

  • Agents are read-only and expose only health and temperature endpoints.
  • /temperature requires the agent API key.
  • Use the optional IP allowlist whenever the Thermo server has a stable IP.
  • Firewall agent port 8090 so only Thermo can reach it.
  • /health may be public unless protected during setup.
  • Do not expose agent ports to the public internet.
